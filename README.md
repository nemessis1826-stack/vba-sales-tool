# VBA 売上集計ツール

## ■ 概要

Excel VBAを使用して、売上実績を自動集計するツールです。
担当者別・商品別の売上を月次で可視化できます。
担当者マスタより、自由にフラグ設定をおこない集計対象者を選別できます。
（０＝集計対象外、１＝個別集計、２＝インサイドセールスを含めた集計）
---

## ■ 主な機能

* 売上データの自動抽出
* 担当者変更に応じた動的集計
* ボタン1つで集計処理を実行
* レイアウト自動整形

---

## ■ 使用技術

* Excel VBA

---

## ■ 想定利用シーン

* 営業部門の売上管理
* 月次レポート作成の効率化
* 手作業集計の自動化

---

## ■ ポイント

* 手動作業を削減し、業務効率を向上
* 担当者変更にも柔軟に対応可能
* 実務を想定した設計
